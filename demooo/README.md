# 太行残雪 —— 红色历史沉浸式视觉小说
基于1942年十字岭反扫荡史实创作，红色宣传、思政教育、博物馆联动适用

## 项目说明
- 类型：红色历史视觉小说
- 背景：太行山脉 · 十字岭反突围 · 左权将军事迹
- 用途：学校活动、红色宣传、博物馆联动、线下展示

## 文件结构
项目基于 Django 搭建，结构清晰：
- code/ 项目主目录
- main/static/images/ 背景图存放目录
- main/templates/ 页面目录
- main/views.py 逻辑文件


## 功能特点
- 10个剧情抉择节点
- 3种不同结局
- 16张背景图自动切换
- 支持博物馆联动、图文科普
- 纯鼠标操作，零代码、零命令

## 注意事项
- 图片必须放在指定目录，否则无法显示
- 每个文件修改后必须保存
- 运行前确保目录结构完整
- 适用于电脑、大屏、手机端展示
